Contributors: Oturia Web Design
Requires at least: 3.3
Tested up to: 3.4
Stable tag: 1.0
Tags: Google, Analytics, Webmaster Tools

== Description ==
This plugin makes it easy for you to add your Google Analytics tracking code as well as your meta tag verification of Webmaster Tools.

== Plugin Official Page ==
http://oturia.com/plugins/smart-google-code-inserter/

== Installation ==
1. Upload the entire `smartgooglecode` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
